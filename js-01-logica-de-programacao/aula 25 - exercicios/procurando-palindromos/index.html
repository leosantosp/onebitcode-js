Procurando Palíndromos

Escreva um programa em JS que seja capaz de identificar se uma palavra é um palíndromo.
Um palíndromo é uma palavra que lida de ttrás para frente possui as mesmas letras na mesma ordem.
O programa deve iniciar pedindo que seja informada uma palavra e então exibir uma mensagem dizendo se aquela palavra é ou não um palindromo.
Caso não seja, deve mosrar a palavra lida da esquerda para direita e da direita para esquerda.

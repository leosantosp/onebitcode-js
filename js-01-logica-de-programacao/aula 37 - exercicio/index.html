Calculadora Geométrica

Escreva um programa em JS para calcular a área de diferentes formas geométricas. O programa deve iniciar com um menu contendo as diferentes opções de cálculas. As opções devem ser: - Área do Triângulo: base * altura / 2 - Área do Retângulo: base * altura - Área do Quadrado: lado ao Quadrado - Área do Trapézio: (base maior + base menor) * altura / 2 - Área do Círculo: pi * raio2 (considere pi: 3.14) Usar funções para separar os procedimenos. O programa também deve ter opção de 'Sair' e enquanto não for escolhida deverá voltar ao menu.
